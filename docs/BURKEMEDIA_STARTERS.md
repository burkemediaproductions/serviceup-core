# BurkeMedia pilot starters

Two installable packs establish the first pilot without hard-forking ServiceUp:

- `burkemedia-crm`: Clients, Contacts, Leads, Projects, and Tasks.
- `burkemedia-cms`: Blog Posts, Case Studies, Services, and Testimonials.

Install them through Gizmo Packs with distinct Gadget names/slugs. They create reusable Content types; they do not import live BurkeMedia data or modify BurkeMedia.pro.

The Activity area supplies cross-record notes, tasks, and follow-ups. Content relationships and dedicated list/entry views can be refined after real BurkeMedia data exposes the most useful daily workflow.
